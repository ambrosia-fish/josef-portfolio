Meet App

My Meet App is a responsive React application designed to help users discover and manage local events. Key features include: city-based event filtering, collapsible event details, customizable event list size, offline functionality, home screen shortcut for mobile devices, and interactive charts visualizing event statistics. This app combines modern React practices with progressive web app capabilities, offering a seamless and intuitive experience for event enthusiasts.
